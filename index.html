Comissões

Configure a taxa padrão de comissão para Empresas Vigentes

Taxa Padrão Global
Quando definida, esta taxa será aplicada automaticamente ao cadastrar um novo empresa vigente. Você ainda poderá alterar a taxa individualmente em cada Lançamento.
%
Como funciona:
• Ao cadastrar uma entrada com categoria Empresas Vigentes, o campo Comissão será preenchido automaticamente com esta taxa.
• O valor Líquido é calculado como: Bruto − (Bruto × Taxa%)
• Caso nenhuma taxa esteja definida aqui, o sistema exibirá um aviso no formulário solicitando o preenchimento manual.
• A taxa pode ser alterada livremente em cada Lançamento individual.

4. Repita o processo semanalmente

Sistema de Comissões

Cálculo de Comissões

Fórmula:
Valor Bruto × Taxa (%) = Comissão

Exemplo:
R$ 1.000,00 × 10% = R$ 100,00

Líquido:
R$ 1.000,00 - R$ 100,00 = R$ 900,00

Configuração

Taxa padrão:
• Definida em Comissões
• Aplicada automaticamente
• Pode ser alterada por empresa
• Usada em simulações
• Afecta apenas empresas vigentes

Problemas Comuns e Soluções

Esqueci de fazer backup, perdi dados?
Solução: Se você não fez backup, os dados ficam apenas no navegador. Verifique se ainda estão acessíveis. Se não, infelizmente não há como recuperá-los. Recomendação: Faça backup semanalmente automaticamente.
Erro ao importar planilha Excel
Verifique:
• Use o modelo exato fornecido
• Colunas devem ter os nomes corretos
• Formato .xlsx ou .xls
• Valores numéricos sem texto
• Datas em formato válido
• Sem células mescladas
Esqueci minha senha de acesso
Opções:
• Use a função "Redefinir Senha" em Segurança
• Isso apagará todos os dados locais
• Tenha um backup para restaurar depois
• Anote sua nova senha em local seguro
Sistema está lento ou travando
Soluções:
• Limpe cache do navegador
• Feche outras abas/janelas
• Verifique se há muitos dados
• Faça backup e limpe dados antigos
• Use navegador moderno (Chrome/Firefox)
Números de lançamento com sequência estranha
Normal: O sistema usa IDs sequenciais automáticos. Se você importar dados, os IDs podem pular números para evitar conflitos. Isso não afeta o funcionamento do sistema.
